Fast Rates API is free & open source API service for current and historical foreign exchange rates 

            
GET https://fastrates.herokuapp.com/latest HTTP/1.1

Fast & reliable

Fast Rates API has been designed and tested to handle +2000 of request per second.

Open Source

Everything from the code base is opensource and free to use under a permissive MIT license.

Usage

Get the latest foreign exchange reference rates. 

            
GET https://fastrates.herokuapp.com/latest HTTP/1.1

Get historical rates for any day since 2010 

            
GET https://fastrates.herokuapp.com/historical?date=2010-01-12 HTTP/1.1

Rates are quoted against the Euro by default. Quote against a different currency by setting the base parameter in your request. 

            
GET http://fastrates.herokuapp.com/latest?base=USD HTTP/1.1

Request specific exchange rates by setting the symbols parameter. 

            
GET https://fastrates.herokuapp.com/latest?symbols=USD,GBP HTTP/1.1

Get historical rates for a time period. 

            
GET https://fastrates.herokuapp.com/historical?start_at=2010-01-01&end_at=2020-09-01 HTTP/1.1

Quote the historical rates against a different currency. 

            
GET https://fastrates.herokuapp.com/historical?start_at=2018-01-01&end_at=2018-09-01&base=USD HTTP/1.1

Fast Rates is a such a new service it's not supporting all tickers and dates earlier than 2010 yet, but version we are plannig to do that in version 1.5.
